# 실행마케팅 — B2B 마케팅 실행사 웹사이트

Next.js(App Router) + Tailwind CSS + Framer Motion 기반의 다크/라이트 대응 프로덕션 수준 랜딩 사이트입니다.

## 한 번에 실행해서 확인하기

1. **프로젝트 폴더**를 연 다음 **`start.bat`** 파일을 더블클릭하세요.
2. 한글 경로 오류 방지를 위해 프로젝트가 **M:** 드라이브로 연결된 뒤 개발 서버가 실행됩니다.
3. 브라우저에서 **http://localhost:3000** (또는 터미널에 표시된 주소)로 접속하세요.
4. **뷰티/의료, F&B/로컬, 커머스/온라인** 등 메뉴를 눌러도 정상 동작합니다.

또는 터미널에서:

```bash
npm install
npm run dev
```

이후 **http://localhost:3000** 에 접속합니다.

## Internal Server Error / UNKNOWN 오류가 날 때 (뷰티/의료 등 클릭 시)

- **원인**: 프로젝트 경로에 **한글**(예: `마케팅사 홈페이지`)이 있으면 Windows에서 파일 경로 인식 문제로 오류가 납니다.
- **해결**:
  1. **`start.bat`**을 더블클릭해서 실행하세요.  
     → 프로젝트를 **M:** 드라이브로 연결한 뒤 실행하므로, 한글 경로 없이 동작합니다.
  2. 그래도 오류가 나면: 프로젝트 **전체 폴더**를 **영문 경로**로 복사한 뒤 그곳에서 실행하세요.  
     예: `C:\marketing-site` 폴더를 만들고, 지금 프로젝트 파일을 모두 복사한 다음, 그 폴더에서 `npm install` 후 `npm run dev`
  3. OneDrive 동기화 폴더에 두었다면, **.next** 폴더를 동기화 제외하거나, 위와 같이 영문 경로로 복사해 실행하는 것을 권장합니다.

## 빌드 및 배포

```bash
npm run build
npm start
```

Vercel 배포: 저장소 연결 후 자동 빌드되며, 별도 설정 없이 배포 가능합니다.

## 구성

- **메인 (/)** — Hero, 실제 작성 레퍼런스, 플랫폼별 역량(캐러셀), 왜 실행사 직접 연동인가, 고객 후기(캐러셀)
- **뷰티/의료 (/beauty)** — 성형·피부과 타겟, 샤오홍슈·바비톡·강남언니 CSS 목업
- **F&B/로컬 (/local)** — 음식점·카페·미용실 타겟, 캐치테이블·네이버 플레이스·당근 CSS 목업
- **커머스/온라인 (/commerce)** — 스마트스토어·쿠팡·뷰티 타겟, 화해·쿠팡·스마트스토어 CSS 목업

헤더에서 다크/라이트 전환, Footer 문의 폼(이름·연락처·이메일·업종·문의 내용), 우측 하단 챗봇(비용·진행 절차·견적 문의 대화형) 사용 가능합니다.
